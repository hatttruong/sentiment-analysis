HOW TO RUN

1. Setup environment
- Python 3.x
- To install required Python package, under the sentiment-analysis folder, run the following command
    sudo pip3 install Cython
    sudo pip3 install -re requirements.txt

2. How to predict

To install required Python package, under the sentiment-analysis folder, run the following command
    sudo python3 predict.py <input_path> <output_path>
